================
Vagrant Solr Box
================
Solr 7.1 on Ubuntu 16.04 with Banana.

Usage
=====
Install
-------
.. code-block:: sh

	./scripts/install.sh

Start
-----
.. code-block:: sh

	./scripts/start.sh

Stop
----
.. code-block:: sh

	./scripts/stop.sh

Shell
-----
.. code-block:: sh

	./scripts/shell.sh
